LMbench - Tools for performance analysis

How do I get LMbench?

If you only want to run it

Please visit BitMover's ftp site to get the latest and greatest. You can do so by clicking here.

If you want to modify it

LMbench is maintained in BitKeeper; you can get a copy by visiting the link above.

After getting and installing Bitkeeper do the following to get a copy of the source tree and build it. 

$ bk clone http://lmbench.bkbits.net/LMbench2
$ cd LMbench2/src
$ make

In order for the build to complete, the BitKeeper "get" command has to be in your path.

Mailing list

There is a mailing list for LMbench users, it is "lmbench-users@bitmover.com" and you can subscribe by sending a message to majordomo@bitmover.com with the words

subscribe lmbench-users

in the body of the message.
