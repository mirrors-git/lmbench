LMbench - Tools for performance analysis

How do I get LMbench?

If you only want to run it

Please visit lmbench's SourceForge site to get the latest and greatest. 
$ bk clone http://lmbench.bkbits.net/LMbench2
$ cd LMbench2/src
$ make

Mailing list

There is a mailing list for LMbench users, it is "lmbench-users@bitmover.com" and you can subscribe by sending a message to lmbench-users-request@bitmover.com with the words

subscribe lmbench-users

in the body of the message.
